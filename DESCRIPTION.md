# VibeSplash

**⚠️ Vibe coded. Powered by AI. Born from chaos. Surprisingly functional.**

VibeSplash is a Playnite addon that displays a full-screen cinematic splash screen when you launch — and now also when you *close* — a game. Built on the shoulders of [Artzox's Playnite-Splash-Addon](https://github.com/artzox/Playnite-Splash-Addon), this fork adds one juicy extra: a splash screen that greets you when you return to Playnite after gaming. Because your library deserves a dramatic entrance both ways.

## What it does

- Shows a full-screen splash overlay using your game's background art and logo (Extra Metadata Loader supported)
- Smooth fade-in / fade-out transitions
- **NEW:** Optional splash screen triggered *after* closing a game — a cinematic "welcome back" moment
- Fully configurable: global duration, per-platform timers, per-game overrides, excluded games
- Logo size control for that perfect aesthetic

## Honest disclaimer

This addon was **vibe coded** — meaning an AI did the heavy lifting while a human pointed and said "yeah, do that." It works. It might have edge cases. It definitely has vibes. If something's weird, open an issue and we'll vibe-debug it together.
